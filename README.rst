SimpleGNN
=========

A simple way to run predefined and new custom GNNs on predefined and new custom benchmark dataset. The goal is to achieve a good comparison quality between different architectures.

Installation
------------

.. code-block:: bash

    pip install -e .

Usage
-----

.. code-block:: python

    from sample import core
